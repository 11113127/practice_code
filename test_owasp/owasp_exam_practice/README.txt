====================!重要!考試注意事項!====================

考試前應該要準備好的:
    
1. 可供php 或 jsp 或 asp.net語言運行測試的環境
    
2. 可供mysql運行測試的環境



考試時可以做的事情:
    
1. Google
    
2. 翻閱參照任何程式、筆記
    
3. 使用自己帶來的NB



考試時不可進行的事項:
    
1. 拿手機，請關靜音
    
2. 任何形式的交談、討論
    
3. 任何形式的線上交談、討論

如屢勸不聽，關閉考生網路連線，情節嚴重關閉整間網路
